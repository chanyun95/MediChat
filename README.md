# 프로젝트 소개
비대면 진료 서비스🩺<p>
환자와 의료 전문가 간에 원활한 비대면 의료 서비스를 제공하는 웹사이트 입니다.<br>
환자들이 병원 외부에서도, 의료 관련 서비스를 안전하고 효율적으로 받을 수 있도록 설계하였으며,<br>
편리하고 신뢰할 수 있는 비대면 진료 경험을 제공합니다.
